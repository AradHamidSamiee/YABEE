YABEE
=====
Renewed Egg exporter for the Blender 2.8

Exporting:
- Meshes
- UV layers
- Materials and textures (Initial PBR)
- Armature (skeleton) animation
- ShapeKeys (morph) animation
- Non cyclic NURBS Curves
